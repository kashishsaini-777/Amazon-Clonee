# 🛒 Amazon Clone

A full-stack Amazon Clone built using the MERN stack (MongoDB, Express, React, Node.js). This project replicates core e-commerce functionalities like product listing, cart management, authentication, and order handling.

---

## 🚀 Features

- 🔐 User Authentication (Login/Register)
- 🛍️ Product Listing & Details Page
- 🛒 Add to Cart / Remove from Cart
- 💳 Order Placement System
- ⭐ Product Ratings & Reviews
- 📦 Backend API Integration
- 🌐 Fully Responsive UI

---

## 🛠️ Tech Stack

### Frontend:
- React.js
- Axios
- CSS / Tailwind (if used)

### Backend:
- Node.js
- Express.js
- MySQL
- JWT Authentication
